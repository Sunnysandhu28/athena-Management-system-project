GitHub Upload Instructions
1
Download the ZIP file

Click the "Download ZIP Archive" button to save the file to your computer.

2
Access your GitHub repository

Go to your Cathena project repository on GitHub.

3
Upload the file

Click "Add file" > "Upload files", then drag and drop the ZIP file.

4
Commit the changes

Add a descriptive commit message and click "Commit changes".

5
Extract the files

To extract the ZIP file in GitHub, click on it and follow the extraction prompts.

Alternative Approach

You can also extract the ZIP file locally and manually upload each individual template file to the appropriate location in your GitHub repository.
